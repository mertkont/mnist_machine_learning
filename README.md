# Machine Learning Applications on MNIST Data Set

Machine learning applications on MNIST data set. The MNIST (Modified National Institute of Standards and Technology database) dataset contains a training set of 60,000 images and a test set of 10,000 images of handwritten digits. The handwritten digit images have been size-normalized and centered in a fixed size of 28Ã—28 pixels.
